# ScalperToday API Reference

## Base URL

```
# Desarrollo
http://127.0.0.1:8000/api/v1

# Produccion
https://scalpertoday-ruben.azurewebsites.net/api/v1
```

## Authentication

Endpoints protected with JWT require the header:
`Authorization: Bearer <your_token>`

---

## 1. Events

### GET `/macro`
Returns the economic calendar for today with AI analysis.

**Response:** `List[EventResponse]`
```json
[
  {
    "id": "event_123",
    "time": "14:30",
    "title": "Non-Farm Payrolls",
    "country": "United States",
    "currency": "USD",
    "importance": 3,
    "actual": "225K",
    "forecast": "180K",
    "previous": "150K",
    "surprise": "positive",
    "ai_analysis": {
      "summary": "Strong job growth...",
      "impact": "HIGH",
      "sentiment": "BULLISH"
    }
  }
]
```

**Possible errors**
- `503`: no events available from provider and no cached events for today.

### POST `/macro/refresh`
Forces a fresh refresh from the configured calendar provider.
- **Header required**: `X-API-Key: <refresh_key>`

## 2. Briefing & Summary

### GET `/brief`
Returns a comprehensive market outlook generated by AI.

### GET `/home/summary`
Returns data optimized for the mobile dashboard (greeting, stats, next event, highlights).

---

## 3. Alerts

### GET `/alerts`
List all user alerts.

### POST `/alerts`
Create a new alert.
**Body:**
```json
{
  "name": "USD High Impact",
  "conditions": [
    {"alert_type": "importance", "value": "3"},
    {"alert_type": "country", "value": "United States"}
  ],
  "push_enabled": true
}
```

### POST `/alerts/device-token`
Register a mobile device for push notifications.
**Body:**
```json
{
  "token": "ExponentPushToken[xxx]",
  "device_type": "ios",
  "device_name": "iPhone 15"
}
```

---

## 4. Auth

### POST `/auth/register`
Create a new account.

### POST `/auth/login`
Get a JWT token.

### GET `/auth/me`
Get current user profile.
